ChromNet is designed to identify direct interactions between ChIP-seq datasets, which are correlations unique to a pair of variables that cannot be explained by any other dataset in the database. ChromNet simultaneously displays both a hierarchical clustering of the data represented by color intensity, and direct interactions between groups represented by edges. For more details and raw data see paper to come. Source code for both the client interface and to generate custom ChromNet networks is available on GitHub.
